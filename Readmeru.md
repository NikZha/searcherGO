# SearcherGO — лёгкий поисковый движок для email

SearcherGO — это утилита для массового поиска email-адресов по HTML-файлам и сохранённым страницам из поисковых систем (Google, Яндекс и любых других).

## 🎯 Зачем это нужно?

Поиск контактов на сайтах — это боль:
- **Поисковики** — реклама, капчи, ограничения
- **Корпоративные сайты** — информация устаревшая или недостоверная
- **Звонки** — бесконечные вопросы, ожидание на линии, отсутствие нужного человека

**Email решает всё.** Одно грамотно составленное письмо даёт ответ быстрее, чем час обзвона.

Но кликать по каждой ссылке, искать email и повторять цикл — долго и скучно. SearcherGO автоматизирует этот процесс: вы загружаете HTML-файл со ссылками, а программа находит все email за секунды.

## 🚀 Быстрый старт

### Требования
- Go 1.21+ (для сборки)

### Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/NikZha/searcherGo
cd searcherGo

# Собрать бинарник
go build -o searcher.exe

# Запустить
./searcher.exe
```
Или просто дважды кликните start.bat — программа откроет консоль с диагностикой и автоматически закроет её при завершении работы.

После запуска откройте браузер и перейдите по адресу написанному в консоли.

## 📦 Что внутри?

| Было (Java/Spring) |	Стало (Go) |
 ------------- | ------------- |
| 50+ МБ бинарник + JVM |	10 МБ один файл |
| Сохранение в БД (невостребованная фича) |	Убрано — простота важнее |
| Поиск через URL-запросы (не работает из-за капч)|	Отказ от этого подхода |
|Сервер на Spring Boot |	Старт за миллисекунды|

## 💡 Как это работает?

1. Вы сохраняете HTML-файл с результатами поиска (Google, Яндекс, любой другой)

2. Загружаете его в веб-интерфейс

3. JS собирает все ссылки вида ```<a href="...">```

4. Бэкенд на Go обрабатывает каждую ссылку и извлекает email

5. Результат возвращается в виде таблицы
